Methodology

Each poll is weighted by its sample size, multiplied by a population factor — 1.5x for likely voters, 1.25x for registered voters, and 1x for all other populations. A 15-poll rolling weighted average is then applied to smooth short-term noise while staying responsive to real shifts in opinion. This mirrors the same pipeline used in the original Python analysis, ported to run live in the browser on every page load.
